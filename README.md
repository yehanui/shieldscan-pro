# ShieldScan - Premium URL Security Analysis Platform

A production-ready, futuristic cybersecurity SaaS landing page for URL security analysis. Built with pure HTML5, Tailwind CSS, and Vanilla JavaScript.

## Features

- **3-Second Loading Animation**: Visual feedback with progress bar and rotating shield
- **Direct VirusTotal Redirect**: Automatically opens VirusTotal in new tab after analysis
- **Client-Side URL Validation**: Real-time validation with error handling
- **URL Safety Education**: Comprehensive guide on how to recognize suspicious URLs
- **Privacy-First Design**: No data storage, no API keys required, all client-side
- **Premium UI/UX**: Glassmorphism, futuristic design, smooth animations
- **Mobile-First Responsive Design**: Perfect on all devices
- **Zero Backend Dependencies**: 100% frontend, ready for static hosting

## How to Use

1. Enter any URL in the input field
2. Click "Analyze URL" or press Enter
3. Watch the 3-second loading animation with progress bar
4. Automatically redirects to VirusTotal for comprehensive malware scanning
5. Learn to recognize suspicious URLs in the educational section

## Workflow

1. **URL Input**: User enters URL
2. **Client-Side Validation**: Real-time validation with visual feedback
3. **3-Second Animation**: Loading animation with progress indicator
4. **VirusTotal Redirect**: Opens VirusTotal in new tab with encoded URL
5. **UI Reset**: Form resets automatically for next analysis

## VirusTotal Integration

- **Base64 Encoding**: URLs are encoded for VirusTotal API
- **Secure Redirect**: Opens in new tab with `noopener,noreferrer` security attributes
- **No Data Storage**: URLs processed client-side only
- **Error Handling**: Graceful error messages for invalid URLs

## URL Safety Guide Features

- 6 visual tips for spotting malicious URLs
- Quick reference table of red flags
- Pro tips for URL safety
- Real examples of suspicious vs safe URLs

## Security & Privacy

- **No Data Storage**: URLs never leave the browser
- **No Tracking**: Zero analytics or user tracking
- **No Backend**: All processing happens client-side
- **Transparent**: Clear explanations of how URL analysis works

## Disclaimer

ShieldScan provides client-side URL validation and redirects to VirusTotal for comprehensive analysis. We do not guarantee complete protection against all threats. Always exercise caution when visiting unfamiliar websites and use comprehensive security solutions.
